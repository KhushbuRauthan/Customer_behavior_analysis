# Customer_behavior_analysis
OVERVIEW

This project analyzes retail customer shopping behavior to identify purchasing patterns, revenue trends, and customer segments. The analysis involves data preprocessing in Python, running analytical queries in MySQL, and building an interactive dashboard in Microsoft Power BI.
________________________________________________________________________________________________________________________________________________________________
DATASET
The dataset contains retail transaction records including customer demographics, purchase amounts, product categories, and shipping information. It was used to explore customer behavior and generate business insights.
_________________________________________________________________________________________________________________________________________________________________
TOOLS
	•	Python (Pandas, Jupyter Notebook)
	•	MySQL for SQL analysis
	•	Power BI for data visualization
	•	Git & GitHub for version control
_________________________________________________________________________________________________________________________________________________________________
STEPS
	1.	Data Loading – Imported the dataset using Python and explored its structure.
	2.	Data Cleaning – Standardized column names, handled missing values, and prepared data for analysis.
	3.	Exploratory Data Analysis (EDA) – Analyzed distributions, customer segments, and purchasing patterns.
	4.	SQL Analysis – Executed queries to calculate revenue trends, top products, and customer segmentation.
	5.	Dashboard Creation – Built an interactive Power BI dashboard with visual insights.
_______________________________________________________________________________________________________________________________________________________________
DASHBOARD
The Power BI dashboard includes visualizations for:
	•	Sales and revenue trends
	•	Customer demographics
	•	Product performance
	•	Purchase behavior insights
_________________________________________________________________________________________________________________________________________________________________
RESULT
The analysis helped identify key customer segments, top-performing products, and revenue trends that can support data-driven decision making in retail.
_________________________________________________________________________________________________________________________________________________________________
HOW TO RUN
	1.	Clone this repository.
	2.	Open the Jupyter Notebook to view Python data cleaning and EDA steps.
	3.	Import the dataset into MySQL to run SQL queries.
	4.	Open the Power BI file to explore the interactive dashboard.
